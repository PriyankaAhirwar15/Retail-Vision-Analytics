## 📌 Project Overview
This project is an **AI-powered Retail Intelligence System** designed to help store managers track customer footfall in real-time. It uses Computer Vision to detect people and faces, providing a scalable API for retail data analytics.

## 🚀 Features
* **Automated Person Detection:** Uses Haar Cascade classifiers for real-time face and person detection.
* **RESTful API:** Built with **FastAPI** to allow easy integration with mobile or web dashboards.
* **Image Processing:** Converts raw retail footage into grayscale and processes it for high-accuracy inference.
* **Professional Backend:** Structured with modular code, virtual environments, and dependency management.

## 🛠️ Tech Stack
* **Language:** Python
* **AI/CV:** OpenCV, NumPy
* **Backend:** FastAPI, Uvicorn
* **Tools:** VS Code, Git, GitHub
